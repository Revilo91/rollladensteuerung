# HACS Logo Files

This directory contains the logo/branding for the Cover Control Advanced Home Assistant integration.

## Files

- **logo.svg** - Scalable Vector Graphics version (recommended for best quality at any size)
- **logo.png** - PNG raster version (512×512 pixels)

## Design Elements

The logo represents the key features of Cover Control Advanced:

1. **Sun with rays** (top-left) - Represents sun position tracking and intelligent automation
2. **Window with frame** (center-right) - Represents the window/cover being controlled
3. **Venetian blind slats** (on window) - Shows automated shading in action with graduated slats
4. **House icon** (bottom-left) - Represents room-based control and home automation
5. **Gear icon** (bottom-right) - Represents automation and intelligent control
6. **Curved arrow** - Shows the connection between sun tracking and cover control

## HACS Integration

HACS automatically displays the logo when:
- The file is named `logo.png` or `logo.svg` in the repository root
- HACS will prefer SVG for better quality across different display sizes

## Colors

- **Sun**: Warm orange/yellow gradient (#FDB813 to #FF8C00)
- **Window/House**: Blue tones (#4A90E2, #5A9BD5, #2E5C8A)
- **Blinds**: Light gray gradient (#E8E8E8 to #B0B0B0)
- **Accents**: Professional dark blue (#2E5C8A)
